\#  Web Scraper – Fake Job Listings



Dieses Projekt zeigt einen vollständigen Web-Scraper in Python, der automatisch Stellenanzeigen von der öffentlich verfügbaren Demo-Seite  

\*\*https://realpython.github.io/fake-jobs/\*\*  

abruft, relevante Informationen extrahiert und anschließend alle Jobs filtert, deren Titel ein bestimmtes Stichwort enthalten.



Der Scraper dient als Beispiel für Web-Automatisierung, Datenextraktion und Datenverarbeitung mit Python.



---



\##  Funktionen



\- Sendet eine HTTP-Anfrage an eine Job-Webseite

\- Extrahiert:

&nbsp; - Jobtitel  

&nbsp; - Unternehmen  

&nbsp; - Standort  

&nbsp; - Bewerbungslink  

\- Filtert alle Jobs, deren Titel ein bestimmtes Stichwort enthalten (z. B. „Python“)  

\- Speichert die Ergebnisse automatisch als \*\*CSV-Datei\*\*  

\- Klare, gut lesbare Code-Struktur  

\- Nutzung einer stabilen Demo-Jobseite (ideal für Lern- \& Portfoliozwecke)



---



\##  Verwendete Technologien



\- \*\*Python\*\*

\- \*\*Requests\*\* – lädt HTML von Webseiten

\- \*\*BeautifulSoup4\*\* – extrahiert Inhalte aus HTML

\- \*\*pandas\*\* – speichert Daten in Tabellenform (CSV)
